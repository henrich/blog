まだdebootstrapいじってる

Tue, Jun 26, 2018 in floss using tags debian , debootstrap

そういえば、1.0.100, 1.0.101, 1.0.102, 1.0.103, 1.0.104 という感じでdebootstrap直したりバグ埋め込んだりしてましたね。

この中で得た知見…

  • systemdのcontainer環境下かどうかは $container を見ればいい、あるいはsystemdが動作している環境なら /var/run/systemd/container 見る
  • dnsmasq を使うと Proxy DHCP が簡単にできるので、既にLAN内でDHCPサーバーが動いている環境でPXEブートしてみたい、という時に便利（参考にさせてもらったサイト)
